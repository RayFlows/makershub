# config

运行时配置目录，负责把 `.env`、环境变量和默认本地开发值整理成统一的 `Settings`。

本目录负责：

- 应用基础信息、环境标识和 API 前缀；
- JWT、微信、小程序、邮箱、数据库、日志、MinIO 和 CORS 配置；
- 把逗号分隔、布尔值、数值等环境变量转换为业务代码可直接使用的 Python 类型；
- 为本地开发提供安全的默认值；
- 在生产环境启动时拒绝明显的本地占位密钥和默认对象存储凭据。

本目录不负责：

- 业务规则判断；
- 密钥托管；
- 生产配置文件生成。

生产环境必须通过环境变量或部署平台 secret 覆盖敏感配置，不能把真实密钥写入仓库。
当前生产环境至少会校验 `JWT_SECRET_KEY`、`MINIO_ACCESS_KEY`、`MINIO_SECRET_KEY`
和 `CORS_ORIGINS`，后续新增敏感配置时也应在 `Settings` 中补充启动期校验。
