# 借用

负责物资借用、场地借用、工位申请、审批、取消、归还、押金冻结和异常处理。

当前第一阶段已经落地物资借用的核心申请、审批、取消和归还能力：

- `models.py`：借用申请、明细、审核记录和归还记录；
- `materials/`：物资借用申请、审批、取消、归还服务；
- `constants.py`：借用类型、状态、审核动作和归还情况常量；
- `types.py`：分页结果对象；
- `utils.py`：借用域文本、数量、动作和状态校验工具。

第一阶段允许不同资源类型保留不同申请字段，但共享生命周期、审计和积分冻结概念。
项目借用必须等项目模块落地后开放；当前 `usage_type=project` 会被明确拒绝。

第一阶段 P0 仍缺少修改待审核/已驳回申请的 HTTP 契约、服务校验和测试。缺少
`PATCH /api/v1/borrowing/applications/{application_id}` 前，不能把物资借用标记为
端到端可验收完成。
