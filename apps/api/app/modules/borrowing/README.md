# 借用

负责物资借用、场地借用、工位申请、审批、取消、归还、押金冻结和异常处理。

当前第一阶段已经落地物资借用的核心申请、修改、审批、取消和归还能力：

- `models.py`：借用申请、明细、审核记录和归还记录；
- `materials/`：物资借用申请、修改、审批、取消、归还服务；
- `constants.py`：借用类型、状态、审核动作和归还情况常量；
- `types.py`：分页结果对象；
- `utils.py`：借用域文本、数量、动作和状态校验工具。

第一阶段允许不同资源类型保留不同申请字段，但共享生命周期、审计和积分冻结概念。
