# 人工积分调整能力

本目录负责受控人工补发和人工扣减积分。

人工调整只服务系统级兜底、异常修复和明确受控场景，不是日常积分规则审批入口。
调用方必须在 HTTP 层或应用用例层完成权限校验和审计日志记录。

## 当前能力

- `manually_adjust_points(...)`：按正负金额补发或扣减积分；
- 必填调整原因；
- 通过 `idempotency_key` 防止重复调整；
- 每次真实调整都追加积分流水。

## 不负责什么

- 不负责判断当前操作者是否有 `points.manual.adjust` 权限；
- 不负责记录审计日志；
- 不负责固定规则、临时规则和任务积分发放。
