# 积分流水能力

本目录负责积分流水追加、分页查询和幂等结果读取。

积分流水是账本事实来源。撤销、追回和异常修正必须追加新流水表达反向变化，不能删除
或覆盖已有流水。

## 当前能力

- 分页查询积分流水；
- 按账户当前余额追加流水快照；
- 根据 `idempotency_key` 返回已有操作结果，防止重复发放、重复冻结或重复扣除；
- `reverse_ledger_entry(...)`：对奖励、普通扣分和冻结转扣除流水追加反向修正流水。

## 不负责什么

- 不直接判断人工调整是否允许；
- 不直接处理冻结生命周期；
- 不直接反向修正冻结和解冻流水，这类状态机必须由 `holds` 能力处理；
- 不直接做 HTTP 权限和审计。
