# 成员网页端

成员网页端用于优先验证 MakersHub 的核心业务逻辑。

第一阶段重点覆盖登录、个人资料、积分流水、资源浏览、借用申请、项目立项和项目材料等功能。

## 当前已接入

- 邮箱密码登录；
- 首次邮箱验证码登录；
- 首次设置密码；
- access token 启动校验；
- refresh token 自动续签；
- 退出登录；
- 成员工作台首页；
- 积分余额和最近流水；
- 工作台任务列表、可领取任务、任务发布、提交完成材料和发布人审核。

本地开发默认连接：

```text
http://127.0.0.1:8000/api/v1
```

如需覆盖 API 地址，可设置：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000/api/v1
```
